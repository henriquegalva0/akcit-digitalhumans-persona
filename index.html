Persona Score (Pscore) — Ferramenta de Anotação

Esta ferramenta guia você pelo preenchimento completo de uma avaliação de Persona Score. Siga cada etapa com atenção.

Etapa 0 — Autenticação

Informe o passcode de acesso para liberar o formulário.


Passcode incorreto. Tente novamente.

Etapa 1 — Descrição da Interação

Escreva uma breve descrição do contexto da conversa avaliada (ex: "Persona de filósofo grego respondendo perguntas existenciais").


Etapa 2 — Número de Turnos

Informe quantos turnos de conversa a interação avaliada possui. Cada turno corresponde a um par (pergunta do usuário + resposta do agente). Após confirmar, os campos por turno serão gerados abaixo.


Etapa 3 — Pesos das Categorias

Defina os pesos (w) de cada dimensão de análise. A soma dos quatro pesos deve ser igual a 1 (ex: 0.25 cada). Esses pesos multiplicam as notas Likert de cada categoria no cálculo do Pscore Instantâneo por Turno (Pt).

As categorias são: S (Aderência Estilística), C (Consistência Contextual), R (Resiliência Adversária), O (Estabilidade OOD).









⚠️ Atenção: a soma dos pesos deve ser 1.00.

Etapa 4 — Avaliação por Turno

Para cada turno, informe: a mensagem enviada pelo usuário, a resposta do agente, e as notas Likert (1–5) de cada categoria.

Escala Likert: 5 = Exemplar | 4 = Sólido | 3 = Neutro/Instável | 2 = Falha Parcial | 1 = Colapso

Os campos de turno aparecerão após você confirmar o número de turnos na Etapa 2.

Etapa 5 — Metadados da Conversa

URL da conversa: Cole o link direto da conversa avaliada (ex: link de um chat no Claude.ai, ChatGPT, etc.).




Modelo: Informe o nome do modelo de linguagem utilizado (ex: claude-opus-4, gpt-4o, gemini-1.5-pro).




Persona Prompt (instruções): Cole apenas as instruções da persona — sem incluir o primeiro comando/interação. Este campo descreve o comportamento que o modelo deveria seguir.


Etapa 6 — Cálculo e Envio

Ao clicar em "Calcular e Enviar", o sistema irá:

  1. Calcular os Pscores Instantâneos por Turno (Pt) usando pesos e notas Likert;
  2. Calcular a Taxa de Erosão (λ) via regressão linear (mínimos quadrados);
  3. Calcular o Pscore Final com decaimento exponencial;
  4. Enviar tudo em uma única linha para o Google Sheets.
